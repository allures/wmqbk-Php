#### 介绍

无名轻博客v4.1.1正式版发布
运行环境：PHP>=5.6 pdo_sqlite

#### 更新如下：

- 1、见readme.txt

#### 使用前请执行安装程序


- 1、密码：安装时随机生成
- 2、如果从V3.0升级请运行v3tov4.1.1.php
- 3、如果从v4升级请运行v4to4.1.1.php
- 4、全新安装请删除v3tov4.1.1.php、v4to4.1.1.php
